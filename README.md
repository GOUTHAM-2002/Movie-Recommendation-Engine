## What It Does

Recommends movies based on:
- **User ratings** (collaborative filtering)
- **Movie content** (TF-IDF similarity)

```bash

# Run notebook
jupyter notebook
```

Open `MovieRecomendation(Collaborative filtering).ipynb` and run all cells.

---

## How It Works

1. **Collaborative Filtering**: Finds users with similar tastes and recommends their favorites
2. **TF-IDF**: Finds movies with similar descriptions/genres
3. **Hybrid**: Combines both for better results
